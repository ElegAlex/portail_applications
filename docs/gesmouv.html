1. Introduction

Bienvenue sur la page de documentation de l'application GESMOUV (version 2.6). Cet outil a été conçu pour la gestion des actions liées aux mouvements du personnel au sein de l'entreprise (arrivées, départs, mutations, changements de site géographique). Cette documentation vous guidera à travers ses fonctionnalités et son utilisation.

2. Fonctionnalités Clés

  • Suivi centralisé des actions à réaliser lors d'un mouvement de personnel.
  • Notification automatique des acteurs concernés (personnel en mouvement, responsables hiérarchiques, services support).
  • Tableau de bord synthétique pour visualiser l'avancement des actions.
  • Validation et contrôle des actions réalisées.
  • Possibilité pour les services support de compléter les descriptions des actions (ex: liste de matériel à restituer).

3. Public Cible

L'application GESMOUV est destinée principalement aux :

  • Personnel en partance ou rejoignant l'entreprise.
  • Responsables hiérarchiques.
  • Responsables des services support impliqués dans les mouvements de personnel (ex: RH, Informatique).

4. Prérequis

Avant d'utiliser GESMOUV, assurez-vous de disposer des éléments suivants :

  • Avoir reçu un email automatique de l'application GESMOUV (anciennement GESDEP) vous notifiant d'un mouvement et contenant un lien d'accès.
  • Le mouvement doit avoir été initialisé par les services RH.

5. Guide d'Utilisation

Ce guide vous explique pas à pas comment utiliser les fonctionnalités principales de GESMOUV.

5.1. Personnel en partance ou rejoignant l'entreprise

Si vous êtes concerné par un mouvement (arrivée, départ, mutation, changement de site), vous recevrez un email de GESMOUV.

  1. Accès à l'application : Ouvrez l'application en cliquant sur le lien présent dans l'email reçu.
  2. Tableau de synthèse : Un tableau de synthèse s'affiche, indiquant le nombre d'actions restant à réaliser pour votre mouvement. Une case rouge indique des actions restantes, une case verte signifie que tout est terminé.
  3. Détail des actions : Cliquez sur la case indiquant le nombre d'actions (ex: "1/7") pour ouvrir le tableau décrivant les actions à réaliser.
  4. Informations complémentaires : Certaines actions peuvent avoir des informations complémentaires (indiquées par une case "verte" ou une icône spécifique dans le manuel). Consultez-les.
  5. Validation des actions : Pour chaque action terminée, cochez la case "action réalisée". Si une action ne vous concerne pas, cochez la case "NC" (non concerné). Les actions validées apparaissent sur fond bleu, et la date et votre nom sont enregistrés.

5.2. Responsable Hiérarchique & Responsable d'un Service Support

Si vous êtes un responsable hiérarchique ou un responsable d'un service support, vous recevrez un email vous informant du mouvement d'un personnel.

  1. Accès à l'application : Ouvrez l'application via le lien dans l'email.
  2. Tableau de synthèse global : Un tableau de synthèse s'affiche avec tous les mouvements pour lesquels vous avez des actions à réaliser, contrôler ou compléter.
  3. Compléter la description des actions (si applicable) :
    • Cliquez sur une case dans la colonne "Nb actions complétées" pour le personnel concerné.
    • Vous verrez la liste des "Actions à Compléter".
    • Pour une action, sélectionnez le symbole "vert" (ou icône dédiée) pour ouvrir une boîte de dialogue.
    • Saisissez les informations complémentaires (ex: liste de matériel à restituer pour le service informatique).
    • Validez avec le bouton "Modifier".
  4. Réalisation des actions (si sous votre responsabilité) :
    • Cliquez sur une case dans la colonne "Nb actions réalisées" pour le personnel concerné.
    • Validez chaque action terminée en cochant "action réalisée" ou "NC". Les actions déjà réalisées apparaissent sur fond bleu.
  5. Contrôle des actions :
    • Cliquez sur une case dans la colonne "Nb actions contrôlées" pour le personnel concerné.
    • Vous verrez la liste des "Actions à contrôler". Les lignes sur fond bleu sont celles déjà réalisées par l'agent ou un autre acteur.
    • Cochez la case "Action contrôlée". La date et votre nom sont automatiquement renseignés.
    • Si vous êtes responsable à la fois de l'action et de son contrôle, la validation du contrôle valide les deux opérations.

Important : Il est crucial de compléter la description des actions et de valider/contrôler les actions en temps voulu pour assurer la fluidité du processus de mouvement du personnel.

6. Dépannage / FAQ

Question 1 : Je n'ai pas reçu d'email de GESMOUV, que faire ?

Réponse : Assurez-vous que votre mouvement a bien été initialisé par les services RH. Si c'est le cas et que vous n'avez toujours pas reçu d'email, contactez les services RH ou le support applicatif.

Question 2 : La case d'une action est rouge dans le tableau de synthèse, qu'est-ce que cela signifie ?

Réponse : Une case rouge indique qu'il reste des actions à réaliser, à compléter ou à contrôler pour le mouvement concerné.

Question 3 : Comment savoir si une information complémentaire est disponible pour une action ?

Réponse : Dans le détail des actions, une icône spécifique (souvent une case verte ou un symbole "+") à côté de l'action indique la présence d'informations complémentaires à consulter.
